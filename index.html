Descripción del curso: El curso cubre los fundamentos básicos de los lenguajes HTML, CSS y JavaScript, que son los elementos esenciales de cualquier aplicación web.

Contenidos del curso:

  1. Introducción al HTML

  4. Frameworks de diseño responsivo

  6. El modelo de Objetos de Documentos (DOM) y JavaScript asíncrono
